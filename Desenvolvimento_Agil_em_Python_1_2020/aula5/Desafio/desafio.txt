A Cervejaria "Já Tom MeI - Jáir, Tomás, Mentor e Idanélcio Ltda" 
solicitou para vocês fazerem um programa destinada a uma equipe de 
cadastro e pedidos para a 
I Feira de Cerveja de Ituroró.

Para isso foi determinada algumas especificações:

1 - O programa deve ter um menu interativo com cabeçalho, local para: 
Cadastro de clientes, 
Ver clientes cadastrados, cadastro de produtos, ver produtos cadastrados 
e a opção sair. 

Este menu deve se repetir até que a opção sair for chamada.

2 - Os dados dos clientes, dados dos produtos devem ter funções separadas 
em arquivos 
e os dados devem ser salvos em arquivos separados.

3 - Para cadastro de clientes deve pedir os seguintes dados: Codigo do cliente, 
CPF, Nome completo, data de nascimento, Estado, Cidade, cep, bairro, rua, 
numero da casa, complemento.

4 - Para o cadastro de produtos deve conter os seguintes dados: Codigo da bebida, 
Nome da bebida, tipo da bebida, volume da bebida, preço unitário.


Quem primeiro me entregrar completo e funcionando ganhará um favo de mel!
